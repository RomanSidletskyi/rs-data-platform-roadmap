# Shell Scripting Tasks

## Task 1 - Write A Safe Logging Script

### Goal

Create a small script that appends a timestamp to a log file.

### Requirements

- use bash shebang
- use `set -euo pipefail`
- create the log directory if missing

## Task 2 - Backup One Config File

### Goal

Use shell scripting for a deterministic backup action.

### Requirements

- take one input file
- copy it to a backup directory with timestamp in the name
