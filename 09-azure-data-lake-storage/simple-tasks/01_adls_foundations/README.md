# Simple Tasks: ADLS Foundations

## Task 1: What ADLS Gen2 Is And Is Not

Write a short explanation of what ADLS Gen2 is and what it is not.

Your answer should distinguish ADLS from:

- Blob Storage as a flat object model
- Spark
- Databricks
- a catalog or governance layer

## Task 2: Hierarchical Namespace

Explain why hierarchical namespace matters for analytics platforms.

Then explain what it still does not solve by itself.

## Task 3: Resource-Level Mental Model

Explain the difference between:

- storage account
- container or file system
- directory path
- file

Then explain which design decisions belong at each level.

## Task 4: Better Storage Does Not Mean Better Platform

Explain why moving data into ADLS does not automatically mean the platform is well designed.

## Task 5: Edge Case Foundation Check

A team says, "We already moved everything from local files into ADLS, so governance and platform architecture are now mostly solved."

Explain why this conclusion is weak.

Then name several platform responsibilities that still remain unsolved.
