# Reference Example - Kafka To Databricks Serving Pipeline Lab

This folder contains a ready comparison example for a Databricks serving pipeline built from landed order events.

Its purpose is:

- self-checking after the guided project
- comparing serving-table contract decisions
- preserving a reusable pattern for Databricks gold delivery from event data

You should attempt the guided project first:

- `07-databricks/pet-projects/04_kafka_to_databricks_serving_pipeline_lab`

## What This Reference Example Demonstrates

- a deterministic landed event sample
- a local preview of a serving-oriented gold projection
- a contract checker for job and warehouse-facing delivery settings
