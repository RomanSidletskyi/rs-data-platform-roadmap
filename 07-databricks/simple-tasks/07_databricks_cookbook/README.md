# Databricks Cookbook Tasks

## Task 1 - Build A Compute Decision Rule

### Goal

Turn compute selection into a repeatable heuristic.

### Requirements

- list the questions you would ask before choosing all-purpose, job, or warehouse compute

## Task 2 - Build A Governance Rollout Rule

### Goal

Capture a safe Unity Catalog rollout pattern.

### Requirements

- define the order in which you would decide domains, objects, and permissions

## Task 3 - Build A Cost Repair Rule

### Goal

Create a small runbook for rising Databricks spend.

### Requirements

- list the first five signals you would inspect
- explain how those signals separate platform misuse from real capacity need
