# Databricks Simple Tasks

These tasks mirror the seven learning blocks in `learning-materials/`.

The goal is to make the learner explain Databricks decisions in small focused exercises before moving into larger lakehouse labs.

## Task Blocks

1. `01_databricks_foundations`
2. `02_workspace_compute_and_runtime`
3. `03_data_engineering_patterns`
4. `04_unity_catalog_governance_and_security`
5. `05_databricks_sql_and_analytics_serving`
6. `06_operating_databricks_platforms`
7. `07_databricks_cookbook`

## Recommended Order

Use tasks immediately after the matching theory block.

The goal is not only to recall terminology.

The goal is to force the learner to defend compute, governance, and delivery choices in a platform context.
