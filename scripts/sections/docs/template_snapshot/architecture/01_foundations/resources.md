# Learning Resources - Foundations

## How To Use These Resources

Use this file as a reading map, not a dump of links.

When reading any foundation resource, extract three things:

- the core distinction being explained
- one real system consequence of misunderstanding it
- one example from this repository where it matters

## What To Search For

- OLTP vs OLAP
- batch vs stream processing
- schema-on-read vs schema-on-write
- ETL vs ELT architecture
- data lineage fundamentals
- idempotent pipeline design

## Best Resource Types

- beginner architecture explainers for terminology
- vendor-neutral engineering articles for trade-offs
- architecture talks that compare batch and streaming honestly

## How To Read Articles Well

Do not only collect definitions.

Ask:

- what design decision becomes easier after understanding this concept
- what anti-pattern becomes visible after understanding this concept
