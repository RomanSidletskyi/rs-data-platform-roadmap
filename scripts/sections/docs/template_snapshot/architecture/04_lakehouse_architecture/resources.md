# Learning Resources - Lakehouse Architecture

## How To Use These Resources

Use lakehouse resources to understand system layering, not only vendor features.

After each resource, note:

- what problem the bronze, silver, and gold split is solving
- why table formats matter operationally
- where storage, compute, and BI boundaries are drawn

- lakehouse architecture overview
- Delta Lake concepts
- medallion architecture guidance
- bronze silver gold design
- open table format trade-offs
