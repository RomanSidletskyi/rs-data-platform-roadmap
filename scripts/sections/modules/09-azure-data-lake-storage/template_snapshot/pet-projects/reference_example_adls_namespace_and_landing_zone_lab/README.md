# Reference Example ADLS Namespace And Landing Zone Lab

This folder is a solved reference example for the guided project:

- `09-azure-data-lake-storage/pet-projects/01_adls_namespace_and_landing_zone_lab`

## What A Strong Solution Should Demonstrate

- clear raw, curated, and publish boundary intent
- stable domain-aware paths instead of vague folder names
- explicit distinction between internal-only and consumer-facing locations
- one explanation of why publish paths are not just renamed curated folders

## Recommended Contents

- boundary rationale
- storage blueprint
- notes on ownership and path stability
- checks for accidental consumer coupling
