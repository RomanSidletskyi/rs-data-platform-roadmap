# 01 Personal Repo Hygiene Lab

## Project Goal

Build a guided personal repository hygiene workflow.

## Scenario

You have a messy repository with untracked files, noisy diffs, weak ignore rules, and oversized commits. The goal is to make the workflow clean and reviewable.

## Expected Deliverables

- cleanup plan
- `.gitignore` strategy
- example commit plan
- verification checklist using status and diff commands

## Starter Assets Already Provided

- `.env.example`
- `src/setup_dirty_practice_repo.sh`
- `data/sample_git_status_before.txt`
- `tests/fixture_expected_cleanup_checklist.txt`

## Definition Of Done

The repository is cleaner, reviewable, and easier to operate safely.
