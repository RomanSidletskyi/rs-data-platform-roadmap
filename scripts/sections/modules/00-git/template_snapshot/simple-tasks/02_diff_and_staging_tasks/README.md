# Diff And Staging Tasks

## Task 1 - Compare Unstaged And Staged Changes

### Goal

Understand what lives in the working tree versus staging area.

### Requirements

- modify a tracked file
- inspect `git diff`
- stage it
- inspect `git diff --staged`

## Task 2 - Build Two Small Commits Instead Of One Large Commit

### Goal

Practice commit boundaries.

### Requirements

- make two unrelated file changes
- commit them separately

## Task 3 - Inspect One Commit Deeply

### Goal

Use `git show` to inspect exact content of a commit.
