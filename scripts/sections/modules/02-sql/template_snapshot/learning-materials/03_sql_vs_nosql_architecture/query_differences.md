# Query Differences

## SQL

Rich declarative language:

- joins
- window functions
- group by
- subqueries
- CTEs

## MongoDB

Document-oriented:

- find filters
- projections
- aggregation pipeline
- nested fields
- array operations

## DynamoDB

Key-based:

- query by PK/SK
- GSIs
- scans are discouraged
- no joins

## CosmosDB

Document SQL-like:

- JSON field access
- partition-aware behavior
- limited compared to full relational SQL
