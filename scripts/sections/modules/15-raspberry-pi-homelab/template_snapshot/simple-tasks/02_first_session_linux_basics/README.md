## What To Practice

- checking current user, hostname, IP, disk, and memory
- moving between folders
- listing files
- creating and removing files and directories
- editing and saving a small file with `nano`
- rebooting and exiting safely

## Suggested Workflow

1. log in to Raspberry Pi
2. run basic host inspection commands
3. move through several folders with `cd`
4. create a test directory and file
5. edit a small file with `nano`
6. exit the SSH session cleanly

## Definition Of Done

- you can move around the filesystem confidently
- you can create, inspect, and remove basic files
- you know how to reboot and exit the session safely
