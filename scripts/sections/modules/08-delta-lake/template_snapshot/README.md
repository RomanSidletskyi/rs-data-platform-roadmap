# Delta Lake

This module covers Delta Lake as the table reliability layer in modern lakehouse platforms.

The focus is not only on features such as `MERGE`, time travel, or schema evolution.

The focus is also on:

- why Delta Lake exists
- what problems it solves over plain Parquet
- how it behaves inside Spark and Databricks platforms
- where Delta design can still fail even when the technology is correct

Start with the learning materials, then use simple tasks and pet projects to practice the patterns.
