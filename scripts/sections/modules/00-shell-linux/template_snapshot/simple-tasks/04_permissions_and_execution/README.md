# Permissions And Execution

## Task 1 - Make A Script Executable

### Goal

Understand why a shell script may refuse to run.

### Requirements

- create `hello.sh` with a valid bash shebang
- inspect its permissions
- make it executable
- run it

## Task 2 - Compare File Modes

### Goal

Practice common permission values.

### Requirements

- set one file to `644`
- set one script to `755`
- compare with `ls -l`

## Task 3 - Diagnose Permission Denied

### Goal

Write the debugging steps you would use when a script says `Permission denied`.
