# Text Reading And Redirection

## Task 1 - Inspect A File Efficiently

### Goal

Use the right reading command for the job.

### Requirements

- show first 5 lines of `README.md`
- show last 10 lines of `README.md`
- show line count of `README.md`

## Task 2 - Separate stdout And stderr

### Goal

Create one command that produces both normal output and errors.

### Requirements

- redirect stdout to one file
- redirect stderr to another file
- verify both files exist

## Task 3 - Capture Combined Output

### Goal

Write both stdout and stderr into one file.

### Requirements

- run a command that mixes valid and invalid paths
- store all output into one log file
- inspect the last lines of the log
