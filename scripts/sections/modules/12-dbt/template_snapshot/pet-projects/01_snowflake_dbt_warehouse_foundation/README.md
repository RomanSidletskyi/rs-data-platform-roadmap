# Project 01 — Snowflake dbt Warehouse Foundation

## Project Goal

Build a production-shaped dbt warehouse foundation on top of Snowflake raw data.

## Scenario

Your team already lands operational data in Snowflake. The missing layer is a clean analytical transformation project that turns raw orders, customers, and products into trusted marts with tests, lineage, and clear model boundaries.

## Project Type

This folder is a guided project, not a ready solution.

You are expected to build the implementation by following this README and the target design described in `architecture.md`.

If you want to compare your result with a ready implementation afterwards, use the separate reference example in:

- `reference_example_snowflake_dbt_warehouse_foundation`
