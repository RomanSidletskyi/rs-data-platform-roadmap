# Project 03 — dbt CI/CD Multi-Environment Platform

## Project Goal

Build a production-shaped dbt platform setup with multiple environments, warehouse routing, CI validation, and deploy automation.

## Scenario

Your team already knows how to write dbt models, but the repository is not yet operated safely. Developers can overwrite shared schemas, CI is weak, production deploy is unclear, and warehouse usage is too blunt.

## Project Type

This folder is a guided project, not a ready solution.

If you want a ready example afterwards, use the sibling reference project:

- `reference_example_dbt_ci_cd_multi_env_platform`
