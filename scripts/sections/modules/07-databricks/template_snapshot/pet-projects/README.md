# Databricks Pet Projects

These projects should simulate production-shaped Databricks work rather than isolated notebook exercises.

They combine:

- compute selection
- medallion pipeline design
- Unity Catalog governance
- SQL delivery patterns
- jobs, asset bundles, and release thinking

## Project Map

1. `01_medallion_batch_pipeline_workspace_lab`
2. `02_unity_catalog_governed_gold_layer_lab`
3. `03_jobs_and_asset_bundle_release_lab`
4. `04_kafka_to_databricks_serving_pipeline_lab`

## Expected Outcome

By the end of these labs, the learner should be able to reason about Databricks as a governed shared platform with clear compute, data, and delivery boundaries.
