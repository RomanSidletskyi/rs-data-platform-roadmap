# Learning Prompts

Use these prompts to accelerate understanding while keeping ownership of the learning process.

## Best Use Cases

- simplify a difficult concept without losing precision
- compare similar technologies
- generate quiz questions or self-check prompts
- turn theory into practice questions

## Example Prompts

### Explain Clearly Prompt

"Explain this concept for a data engineer who understands the basics but keeps confusing these two ideas. Use one concrete example and one counterexample."

### Compare Prompt

"Compare these two technologies in terms of architecture role, strengths, failure modes, and when not to use each one."

### Self-Test Prompt

"Create five short questions that test whether I actually understand this topic, not whether I can repeat definitions."

## Rule

Use AI to sharpen your understanding, then restate the answer in your own words.
