# English For Developers Stack

Use this document to build a practical tool stack rather than relying on one app.

## Recommended Stack By Function

### 1. Understanding Text

Tools:

- ChatGPT for explanation
- DeepL for precise translation when necessary
- Readlang or LingQ-style workflows for assisted reading

Best practice:

- read the original text first
- mark only the hard parts
- ask for explanation, not only translation

### 2. Vocabulary Retention

Tools:

- Anki
- Mochi
- Quizlet

Best practice:

- store phrases and short example sentences
- prefer high-frequency technical language over random vocabulary

Bad example:

- storing isolated words like `throughput`

Better example:

- storing a phrase like `The pipeline throughput dropped after the schema change.`

### 3. Listening And Pronunciation

Tools:

- Language Reactor
- YouTube with transcripts
- ELSA Speak
- Speechling

Best practice:

- replay short technical segments
- shadow the speaker aloud
- compare your recording to the source

### 4. Speaking Practice

Tools:

- ChatGPT voice or similar conversational AI
- speaking tutors or exchange platforms when available

Best practice:

- speak about real technical topics
- explain your project, design choice, bug, or architecture

Prompt example:

```text
Act as a senior engineer in a design review. Ask me follow-up questions about my pipeline architecture in English. Correct only major grammar or clarity issues after I answer.
```

### 5. Writing Practice

Tools:

- ChatGPT for rewriting and structure
- DeepL Write or Grammarly for final polishing

Best practice:

- practice on real artifacts
- rewrite issue descriptions, PR notes, and architecture summaries

Example:

Raw note:

```text
We changed ingestion because old path has many duplicates and difficult recovery.
```

Improved note:

```text
We redesigned the ingestion flow because the previous path produced duplicate records and made recovery difficult after partial failures.
```

## Minimal Viable Stack

If you want to start small, one effective stack is:

- ChatGPT for explanation and speaking
- DeepL for precision
- Anki for retention
- YouTube plus transcripts for listening

That is enough to make visible progress if used consistently.
