# LingQ

LingQ-style tools are useful for extensive reading with recurring vocabulary exposure.

## What It Is

Use this class of tool when you want repeated contact with technical English over longer reading sessions.

## Best For

- extensive reading
- recurring vocabulary exposure
- building reading comfort over time

## Not Good For

- precise pronunciation correction
- architecture reasoning

## Strengths

- supports sustained input over time
- good for reading fluency growth

## Weaknesses

- can stay passive if not paired with output practice

## Main Risks

- mistaking comfortable reading support for full language transfer

## How To Use It Well

- pair reading with short summaries, notes, or speaking

## Primary Modes

- learning

## Rule

Reading tools build input strength best when paired with output tasks.
