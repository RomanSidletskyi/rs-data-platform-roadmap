# How To Learn Faster

Learning faster does not mean skipping the hard parts.

It means reducing wasted effort while keeping understanding high.

## Practical Loop

1. Learn the concept from module notes.
2. Try a task or small implementation yourself.
3. Use AI to explain only the exact gap.
4. Retry the task.
5. Ask AI for review or edge cases after you have a solution.

## Rule

If AI is doing the whole task for you, the speed is fake.

The useful acceleration comes from better feedback loops.
