# AI For Technical Writing

## Goal

Use AI to improve clarity and structure in technical writing without losing engineering precision.

## Good Use Cases

- issue descriptions
- PR summaries
- architecture notes
- trade-off writeups
- meeting or research summaries

## Workflow

1. Draft the content yourself in simple language.
2. Ask AI to improve structure and clarity.
3. Ask it to preserve technical precision.
4. Review every change.
5. Keep your own judgment on scope, certainty, and tone.

## Example Prompt

```text
Rewrite this architecture note to be clearer and more concise. Keep all technical constraints, assumptions, and trade-offs. Do not add claims that are not already present.
```

## Failure Mode To Avoid

Do not let rewriting tools make the text smoother but less precise.
