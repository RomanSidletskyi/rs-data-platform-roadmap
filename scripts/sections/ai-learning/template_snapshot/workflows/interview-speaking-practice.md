# Interview Speaking Practice

## Goal

Use AI to practice speaking about technical work clearly, especially in English.

## Workflow

1. Pick one topic: project, bug, trade-off, architecture, or design decision.
2. Explain it aloud in a short structured way.
3. Ask AI to play interviewer and ask follow-up questions.
4. Ask for feedback on clarity, structure, and the most important language mistakes.
5. Repeat with the improved version.

## Suggested Topics

- explain a pipeline you built
- describe a difficult bug and how you found the root cause
- compare two architectural options
- justify a design trade-off

## Example Prompt

```text
Act as a senior interviewer for a data engineering role. Ask me questions about a pipeline I built, then critique my answers for clarity, structure, and technical precision.
```

## Failure Mode To Avoid

Do not optimize only for fluent speech.

The goal is clear technical thinking expressed well.
