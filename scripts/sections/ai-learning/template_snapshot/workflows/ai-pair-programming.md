# AI Pair Programming

## Goal

Use AI as a coding partner while keeping ownership of design and correctness.

## Workflow

1. Define the exact change you want.
2. Ask AI for the smallest viable implementation or refactor.
3. Review the proposed code before applying it.
4. Run validation.
5. Ask for a second pass only where gaps remain.

## Failure Mode To Avoid

Do not keep accepting more generated code when the architecture is still unclear.
