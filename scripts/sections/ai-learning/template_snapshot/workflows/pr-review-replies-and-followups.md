# PR Review Replies And Follow-Ups

## Goal

Use AI to improve how you respond to review comments and convert them into concrete follow-up work.

## Workflow

1. Read the review comment carefully.
2. Classify it: bug risk, style preference, missing test, or design concern.
3. Draft your own reply first.
4. Ask AI to improve clarity and tone without weakening the technical position.
5. If the comment requires code change, turn it into a concrete action list.

## Example Prompt

```text
Rewrite this review reply to be clearer and more professional while preserving the exact technical point. If the reply is weak, suggest a better structure.
```

## Failure Mode To Avoid

Do not use AI to sound agreeable while avoiding the real technical issue.
