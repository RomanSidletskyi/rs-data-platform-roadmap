# Delta Lake Simple Tasks

These tasks mirror the seven learning blocks in `learning-materials/`.

The purpose is to make the learner explain Delta Lake design choices in small focused exercises before moving into larger labs or projects.

## Task Blocks

1. `01_delta_lake_foundations`
2. `02_table_operations_and_change_patterns`
3. `03_reliability_quality_and_recovery`
4. `04_delta_in_lakehouse_architecture`
5. `05_serving_governance_and_interoperability`
6. `06_operating_delta_lake_tables`
7. `07_delta_lake_cookbook`

## Recommended Order

Use tasks immediately after the matching theory block.

The goal is not only to recall Delta syntax.

The goal is to force the learner to defend table-design, mutation, reliability, and consumer-contract choices in a lakehouse context.
