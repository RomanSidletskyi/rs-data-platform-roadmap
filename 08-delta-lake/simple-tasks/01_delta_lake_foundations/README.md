# Simple Tasks: Delta Lake Foundations

## Task 1: Delta Is And Is Not

Write a short explanation of what Delta Lake is and what it is not.

Your answer should distinguish Delta Lake from:

- Parquet
- Spark
- Databricks
- cloud storage

## Task 2: Transaction Log Mental Model

Explain why a Delta table should be understood as data files plus transaction log rather than as files alone.

Then explain why this matters for history and recovery.

## Task 3: Delta Vs Plain Parquet

You inherit a raw Parquet folder used by several jobs.

Explain what Delta Lake adds if the team converts it into a Delta table.

## Task 4: Time Travel And Schema Evolution

Explain why these two features are useful, but also why they should not be treated as convenience toggles only.
