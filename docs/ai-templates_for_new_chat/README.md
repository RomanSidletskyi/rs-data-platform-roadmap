# AI Templates For New Chat

This directory contains repository-specific context packs for starting a new AI session without rebuilding the same repository framing every time.

Use it when you want the AI to begin with:

- the repository map
- formatting and collaboration rules
- module-generation rules
- starter context for deeper work

## Included Files

- `repo-map.md`
- `ai-context.md`
- `ai-formatting-rules.md`
- `module-generation-rules.md`
- `write-to-new-chat.md`

## How To Use It

1. start with `repo-map.md` when the AI needs repository orientation
2. add `ai-context.md` when the task needs stable project framing
3. add `ai-formatting-rules.md` when output consistency matters
4. add `module-generation-rules.md` when creating or refactoring module content
5. use `write-to-new-chat.md` as the session handoff template

## Rule

These files should reduce session startup friction, not become a second documentation system.
