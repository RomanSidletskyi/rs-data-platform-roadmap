# Readlang

Readlang is useful for assisted reading when you want low-friction lookup without abandoning the original text.

## What It Is

Use it for reading practice where comprehension support should stay close to the source material.

## Best For

- reading technical English with quick lookup
- keeping momentum while reading
- collecting useful phrases during reading

## Not Good For

- long-form explanation
- speaking practice

## Strengths

- low friction during reading
- helps preserve contact with original text

## Weaknesses

- can become crutch-based reading if overused

## Main Risks

- looking up every sentence instead of tolerating some ambiguity

## How To Use It Well

- read the original first
- look up only the blocking parts
- collect only high-value phrases

## Primary Modes

- learning

## Rule

Assisted reading is useful only if the original text remains the main artifact.
