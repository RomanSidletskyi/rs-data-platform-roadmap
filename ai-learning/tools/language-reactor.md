# Language Reactor

Language Reactor-style tools are useful for learning from subtitled video with stronger control over listening and repetition.

## What It Is

Use this class of tool to turn video into active input instead of passive consumption.

## Best For

- watching technical videos with transcript support
- replaying short segments for comprehension
- extracting useful phrases from spoken English

## Not Good For

- writing correction
- structured speaking feedback
- deep architectural explanation on its own

## Strengths

- strong support for active listening
- useful bridge between reading and speaking practice

## Weaknesses

- easy to slip into passive watching
- not enough for pronunciation correction alone

## Main Risks

- watching with subtitles but not producing any reusable output

## How To Use It Well

- replay short segments
- pause and shadow the speaker
- extract phrases you want to reuse

## How Not To Use It

- do not watch long stretches without stopping to process

## Primary Modes

- learning
- speaking

## Rule

Transcript support helps only if it leads to active listening, not passive watching.
