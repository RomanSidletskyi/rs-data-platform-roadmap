# Learn A Module With AI

## Goal

Use AI to learn a module faster without becoming dependent on generated explanations.

## Workflow

1. Read the module README and the first theory materials yourself.
2. Write down what is still unclear.
3. Ask AI only about the exact confusing parts.
4. Solve simple tasks on your own first.
5. Ask AI to review your reasoning, not to replace it.
6. Move into the pet project and use AI for design review, edge cases, and validation ideas.
7. Capture what changed in your understanding.

## Example Prompt

```text
I am studying Delta Lake. Explain the difference between schema enforcement and schema evolution using one realistic data-platform example, one failure mode, and one decision rule.
```

## Failure Mode To Avoid

Do not turn the module into a sequence of AI summaries.

The target is durable understanding, not faster scrolling.
