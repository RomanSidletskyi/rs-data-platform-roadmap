# Kafka Pet Projects

These projects are larger than the simple tasks and are meant to simulate production-shaped Kafka work.

They combine:

- event design
- producer and consumer behavior
- schema and ownership discipline
- operational thinking
- system boundaries across storage and processing layers

## Project Map

1. `01_iot_streaming_lab`

## How To Use These Projects

- read the project README first
- inspect the architecture note before building anything
- treat the `src`, `data`, `config`, `docker`, and `tests` folders as a realistic project scaffold
- document your trade-offs as you implement

## Expected Outcome

By the end of these labs, the learner should be able to reason about Kafka not only as a developer tool, but as a platform component inside real data and operational systems.
