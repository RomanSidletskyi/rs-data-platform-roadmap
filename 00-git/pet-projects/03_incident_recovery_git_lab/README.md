# 03 Incident Recovery Git Lab

## Project Goal

Practice guided Git recovery under stressful repository situations.

## Scenario

You simulate common repository incidents such as lost commits, wrong-branch work, detached HEAD, and destructive reset.

## Expected Deliverables

- recovery playbooks
- command logs for each incident
- explanation of why the recovery worked

## Starter Assets Already Provided

- `.env.example`
- `src/create_recovery_scenarios.sh`
- `data/sample_reflog_notes.txt`
- `tests/fixture_expected_recovery_targets.txt`

## Definition Of Done

The learner can recover from multiple repository incidents without destructive guessing.
