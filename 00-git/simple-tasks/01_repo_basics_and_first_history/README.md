# Repo Basics And First History

## Task 1 - Initialize A Practice Repository

### Goal

Create a clean repository from zero.

### Requirements

- initialize a repository in a temporary folder
- add one README
- make the first commit
- show `git log --oneline`

## Task 2 - Keep Status Clean

### Goal

Practice reading repository state.

### Requirements

- create one untracked file
- inspect with `git status`
- stage it
- commit it
- verify clean status
