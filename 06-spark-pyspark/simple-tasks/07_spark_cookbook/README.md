# Simple Tasks: Spark Cookbook

## Task 1: Join Recipe

Write a short checklist for reviewing a new large join before it enters production.

## Task 2: Output Layout Recipe

You are publishing a dataset mainly queried by `event_date` and `region`.

List the key questions you should ask before choosing output layout.

## Task 3: Skew Recipe

One seller produces 40% of the records in a marketplace pipeline.

Explain why this should be treated as an architectural clue, not only as a tuning annoyance.

## Task 4: Replay Recipe

You need to rebuild three weeks of curated output after a transformation bug.

List the main checks you would do before publishing rebuilt data.
